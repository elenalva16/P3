//PARA EJECUTAR LA APLICACIÓN 
1.- Buscamos el .war (contenido en la carpeta del ejercicio2).
2.- Introducimos el .war seleccionado en la carpeta Apache Tomcat, dentro de la carpeta Webapps. 
3.- Nos situamos dentro de la carpeta ApacheTomcat/bin
4.- Damos permisos a los archivos .sh de la siguiente forma -> chmod +x *.sh
5.- Ejecutamos : ./startup.sh y comenzará a iniciarse ApacheTomcat
6.- Dentro del navegador introducimos la siguiente url para comenzar con nuestra pagina web -> http://localhost:8080/Ejercicio2/index.jsp
